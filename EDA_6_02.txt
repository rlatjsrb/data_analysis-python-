데이터 분석 프로세스
- 수집->정제->탐색->모델->해석

EDA(Exploratory Data Anlaysis)
- EDA(탐색적 데이터 분석)는 본격적인 데이터 분석의 첫 번쨰 단계
- 데이터의 모양과 크기, 분포, 빈 값 등 데이터가 어떠한 형태인지 관찰하고 파악하는 단계
- 데이터로 실험을 하기 전, 데이터를 보다 잘 이해하기 위한 단계

EDA 목적
- 데이터의 오류가 없는지 확인
- 각 데이터의 용도나 설명, 의미를 파악
- 데이터의 개별 속성에 대해 파악

EDA 작업
- 값의 종류 파악(숫자, 카테고리, 순서 등)
- 숫자형 데이터 최소, 최대, 평균 확인
- 이상치 및 누락 유무 확인
- 변수간 관계 분석, 분포 확인
- 합치거나 쪼개서 유의미한지 확인
